Date: 2025-12-12
Purpose: Fix broken ___Total Tickets measure that references non-existent TICKET_COUNT field
Issue: TICKET_COUNT column does not exist in data (each row = 1 ticket)
Solution: Use COUNTROWS function instead

===================================================================
CORRECTED MEASURE: ___Total Tickets
===================================================================

SIMPLE VERSION (Recommended):
-------------------------------------------------------------------
___Total Tickets = COUNTROWS('___Summons')
-------------------------------------------------------------------

This measure:
- Counts all rows in the ___Summons table
- Works correctly in all filter contexts (slicers, visuals, matrices)
- Respects filters applied by TYPE, Month_Year, WG2, etc.
- Each row = 1 ticket, so row count = ticket count


ALTERNATIVE VERSION (Context-Aware):
-------------------------------------------------------------------
___Total Tickets =
    CALCULATE(
        COUNTROWS('___Summons')
    )
-------------------------------------------------------------------

This measure:
- Uses CALCULATE for explicit context handling
- Useful if you need to add additional filter logic later
- Functionally equivalent to the simple version in most cases
